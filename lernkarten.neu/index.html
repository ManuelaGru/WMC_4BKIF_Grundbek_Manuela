✏️ JSON-Lernmaterial bearbeiten

Hier kannst du das Lernmaterial direkt anpassen. Das Format:
topics = Liste von Themen, jedes mit title, icon, description und questions.

📖 JSON-Struktur erklärt

{
  "topics": [
    {
      "title": "🔥 Name des Themas",        // Anzeigename
      "icon": "💡",                          // Emoji-Icon
      "description": "Kurze Beschreibung",  // Kleiner Infotext
      "questions": [
        {
          "question": "Frage?",             // Die Frage
          "options": ["A", "B", "C", "D"],  // 4 Antwortmöglichkeiten
          "correct": 0,                     // Index der richtigen Antwort (0=A)
          "explanation": "Warum A richtig"  // Erklärung nach der Antwort
        }
      ]
    }
  ]
}
